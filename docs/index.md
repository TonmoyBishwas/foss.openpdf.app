# OpenPDF

A completely free, open-source PDF reader and editor for Android. No ads, no
accounts, no tracking, and no internet access — everything happens on your
device.

- [Source code on GitHub](https://github.com/TonmoyBishwas/foss.openpdf.app)
- [Download the latest release](https://github.com/TonmoyBishwas/foss.openpdf.app/releases/latest)
- [Privacy policy](privacy-policy.md)

## Features

View, search, and read aloud PDFs · annotate (highlight, notes, drawing) saved
into the file · fill & sign forms · organize pages (rotate, reorder, delete,
extract, merge, split) · create PDFs from images, camera scans, or text ·
password-protect with AES-256.
